# 🚀 Creation-Terminal (V0.2 Pre-release)

Welcome to **Creation-Terminal**, an open-source, custom command-line interface environment built entirely using Windows Batch script. Designed to elevate the classic terminal experience, it offers modular flexibility, an intuitive layout, and a growing ecosystem for custom terminal tools.

> ⚠️ **Project Status & Compatibility**
> Creation-Terminal is actively under development and optimized exclusively for **Windows** environments. Future sibling projects—like the upcoming **Command Prompt++ Is Now Changing The Name To Command Multiply Or C Multiply **—aim to expand device accessibility down the line.

## 🧰 How to Create Your Own Custom Commands

With the rollout of V0.2, you can use `Terminal.library.tools.bat` to effortlessly build and execute your own custom script variations.

1. **Download** both `Terminal.bat` and `Terminal.library.tools.bat` into the **exact same folder**.
2. **Open** `Terminal.library.tools.bat` inside any preferred text editor (such as Notepad or VS Code).
3. **Write** your custom batch functions, shortcuts, or tools directly inside that file.
4. **Run** `Terminal.bat`. Your brand-new custom commands will integrate and load automatically!
